------- OBJECT -------
    Unordered, key value pairs
    Fast insertion/access,removal  O(1)

    searching O(n)

    - in-built methods -
    Object.keys(bj) : O(n)
    Object.entries(obj) : O(n)
    Object.values(obj) : O(n)
    obj.hasOwnProperty('key') : O(1)

------- ARRAYS -------
    Ordered list


    Access : O(1)
    Searching : O(n)

    Array.shift('elem') : O(n)
    Array.unshift() :O(n)
    Array.push('elem') : O(1)
    Array.pop() : O(1)

    Array.concat() : O(n)

    Array.sort() : O(nlog(n))

    Array.slice() : O(n)
    Array.splice() : O(n)

    forEach/map/filter/reduce : O(n)
